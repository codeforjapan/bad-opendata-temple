## 開発環境の構築手順
```sh
$ yarn install // 依存関係のあるパッケージをインストール
$ yarn develop // ローカルサーバーを立ち上げる
```

```
You can now view `bad-opendata-temple` in the browser.

- http://localhost:8000/

View GraphiQL, an in-browser IDE, to explore your site's data and schema
```
